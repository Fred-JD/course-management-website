## Setup
1. Create Virtual Enviroment (Python 3.7)
    - `python3 -m venv env_name`

2. Activate Virtual Enviroment
    - `source env_name/bin/activate`

3. Install extention
    - `pip install -r requirements.txt`

4. Run Flask
    - `export FLASK_APP=application.py`
    - `flask run`
